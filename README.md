
# Movies APP

This is movies app having some basic features like
ADD, Update, Delete, Sorting and Pagination





## Features
- Login
- Registration
- ADD new Movie
- Update Existing one
- Delete
- Sort Movies
- Filter Movies



## Tech Stack

**Client:** React, Axios, Bootstrap

**Server:** Json Serve
