# Movies APP

This is movies app having some basic features like ADD, Update, Delete, Sorting and Pagination

## Features

- Login
- Registration
- ADD new Movie
- Update Existing one
- Delete
- Sort Movies
- Filter Movies

## Technologies

**Client:** React, Axios, Bootstrap

**Server:** Json Server
